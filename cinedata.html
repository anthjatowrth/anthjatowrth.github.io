CinéData

Construction d’un moteur de recommandation de films : ingestion IMDb/TMDb, pipeline de nettoyage, enrichissement (affiches/résumés), modèle Nearest Neighbors et application Streamlit.

Python Pandas Scikit-learn Nearest Neighbors Streamlit
Repo GitHub Démo Streamlit

Contexte

CinéData vise à faciliter la découverte de films via la recommandation. Le projet s’appuie sur des données publiques issues d’IMDb et de TMDb, afin de construire une base filtrée, enrichie et exploitable.

(Tu peux préciser les critères exacts de filtrage et les volumes une fois que tu veux.)

Objectifs

  • Assembler des datasets hétérogènes (IMDb + TMDb) et fiabiliser les clés de jointure.
  • Nettoyer et filtrer (doublons, valeurs manquantes, outliers, cohérence de genres).
  • Enrichir les films (résumés, posters, métadonnées pertinentes).
  • Créer un modèle de recommandation utilisable dans une app Streamlit.

Pipeline Data

Le pipeline automatise l’import, le nettoyage et la préparation des données : normalisation des colonnes, gestion des valeurs manquantes, filtrage des films selon des critères de qualité, et enrichissement via l’API TMDb.

  • Ingestion : fichiers IMDb + requêtes TMDb
  • Nettoyage : NA / doublons / typage / contrôle de cohérence
  • Filtrage : seuils (votes, popularité, années), selon la stratégie
  • Enrichissement : posters, synopsis, genres, casting (optionnel)

Recommandation ML

J’ai mis en place une approche basée sur Nearest Neighbors : transformation des caractéristiques (TF-IDF / encodages), standardisation si besoin, puis recherche de films similaires par distance.

  • Vectorisation des features (texte + attributs)
  • Mesure de similarité (cosine / euclidienne selon choix)
  • Top-N recommandations explicables
  • Évaluation qualitative (tests utilisateurs) + itérations

Application Streamlit

Une interface simple permet à l’utilisateur de sélectionner un film et d’obtenir des recommandations, avec affichage des posters, résumés et informations principales. L’app sert également de vitrine pour explorer la base filtrée et enrichie.

Recherche

Sélection d’un film via recherche ou liste filtrable.

Recommandation

Top films similaires avec infos clés et synopsis.

Exploration

Accès à une base enrichie (genres, années, popularité…).
